# Instruções para fazer Push no GitHub

O repositório já está configurado e o commit foi criado. 

## Para fazer o push, execute os seguintes comandos no terminal:

```bash
cd glide-store-spark-main
git push -u origin main
```

**Nota:** Se o repositório no GitHub já tiver conteúdo, você pode precisar fazer:

```bash
git pull origin main --allow-unrelated-histories
```

E depois:

```bash
git push -u origin main
```

## Alternativa: Usar GitHub Desktop ou VS Code

1. Abra o GitHub Desktop ou VS Code
2. Faça login na sua conta GitHub
3. O repositório já está conectado, basta fazer o push através da interface
